## Description

Gráfico de linhas mostrando evolução de métricas (spend, conversions, CPA, ROAS) ao longo do período selecionado. Eixo duplo para custo vs. volume.

## Acceptance Criteria

- [ ] Gráfico linha com Recharts
- [ ] Eixo esquerdo: CPA, CPC
- [ ] Eixo direito: conversions, cliques
- [ ] Multi-select de até 4 métricas
- [ ] Hover mostra valores exatos
- [ ] Zoom/pan opcional
- [ ] Granularidade: dia, semana, mês
