# Story 2.2: Campaign, Ad Set & Ad Synchronization

**Epic:** 2 - Meta Ads API Integration
**Status:** 📋 Draft
**Complexity:** Média
**Type:** Feature / Integration

---

## Description

Sincronizar campanhas, ad sets e anúncios da Meta API com banco de dados local. Executado manualmente (botão "Atualizar Agora") e automaticamente a cada 1 hora via pg_cron.

## Acceptance Criteria

- [ ] GET /campaigns com paginação
- [ ] GET /adsets e /ads para cada campanha
- [ ] Upsert em banco (insert ou update)
- [ ] Sincronização manual via endpoint (`/api/sync/campaigns`)
- [ ] Agendamento automático via pg_cron (1x/hora)
- [ ] Log de sincronização com timestamp
- [ ] Tratamento de error (API falha, rate limit)
- [ ] Status da última sincronização visível

## Dependencies

- Story 2.1 (Meta API client)
- Story 1.1 (DB schema)
