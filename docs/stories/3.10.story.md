# Story 3.10: Mobile Responsiveness & UX

**Epic:** 3 - Dashboard & Visualizations
**Status:** 📋 Draft
**Complexity:** Média
**Type:** Feature / UI

---

## Description

Garantir dashboard funcional e utilizável em mobile (iOS/Android). Stack cards, ocultar colunas menos importantes, otimizar navegação.

## Acceptance Criteria

- [ ] Funcional em telas < 768px
- [ ] KPI cards empilhados
- [ ] Tabela com scroll horizontal ou compactada
- [ ] Sidebar colapsada em mobile
- [ ] Gráficos responsivos
- [ ] Touch-friendly (50px min tap target)
- [ ] Testes em iPhone 12 mini e Android
