# Story 2.4: Token Refresh & Expiration Management

**Epic:** 2 - Meta Ads API Integration
**Status:** 📋 Draft
**Complexity:** Baixa
**Type:** Feature / Integration

---

## Description

Implementar renovação automática de access tokens Meta antes de expirar (via refresh_token se disponível, ou reauth). Usuário é alertado se token expirou.

## Acceptance Criteria

- [ ] Refresh automático 1 dia antes de expirar
- [ ] Check de expiration antes de cada API call
- [ ] Alert visual se token expirado (banner)
- [ ] Link para reconectar conta
- [ ] Log de refresh events
- [ ] Fallback: redirect para Meta OAuth se refresh falha

- Story 2.1 (Meta API client)
