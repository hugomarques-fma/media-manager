- [ ] Paginação (50 linhas/página)
- [ ] Sort por qualquer coluna
- [ ] Indicador visual de status (verde/cinza)
- [ ] Score de saúde como barra colorida
- [ ] Botão Pausar/Ativar na linha
- [ ] Clique na linha → detalhe da campanha
