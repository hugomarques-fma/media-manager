# Story 2.3: Daily Metrics Capture & Historical Storage

**Epic:** 2 - Meta Ads API Integration
**Status:** 📋 Draft
**Complexity:** Média
**Type:** Feature / Integration

---

## Description

Capturar métricas diárias de todas as campanhas (impressions, clicks, CTR, CPC, conversions, CPA, ROAS, spend) e armazenar em `campaign_metrics` com granularidade diária.

## Acceptance Criteria

- [ ] Edge Function busca insights via Meta API (diariamente)
- [ ] Métricas armazenadas em `campaign_metrics` table
- [ ] Data granularidade: 1 linha = 1 campanha/1 dia
- [ ] Backfill de histórico (últimos 30 dias)
- [ ] Cálculo de deltas (vs. dia anterior)
- [ ] pg_cron agendado para rodar 1x/dia (noite)
- [ ] Log de erros se API falha

## Dependencies

- Story 2.1 (Meta API client)
- Story 2.2 (Campaigns synced)

## Métricas Capturadas

- impressions, clicks, spend
- CTR = clicks/impressions
- CPC = spend/clicks
- CPM = (spend/impressions)*1000
- conversions, CPA, ROAS
