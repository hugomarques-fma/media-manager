**Epic:** 2 - Meta Ads API Integration

## Description

Implementar tratamento de rate limits (429 responses) com exponential backoff e retry automático. Sistema não quebra quando Meta API rate limits são atingidos.

## Acceptance Criteria

- [ ] Detecta 429 responses
- [ ] Exponential backoff implementado (1s, 2s, 4s, 8s...)
- [ ] Max 5 retries antes de falhar
- [ ] Log de rate limit events
- [ ] Não retorna erro para user, apenas aguarda
- [ ] Testes com mock de 429s
