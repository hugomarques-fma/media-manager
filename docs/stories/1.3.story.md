# Story 1.3: Supabase Auth Setup & User Registration/Login Flow

**Epic:** 1 - Infrastructure & Authentication
**Status:** 📋 Draft
**Complexity:** Baixa
**Type:** Feature / Authentication

---

## Description

Configurar Supabase Auth (email/password), implementar fluxo de registro e login na aplicação Next.js. Usuários conseguem se registrar, fazer login e receber JWT tokens automaticamente.

## Acceptance Criteria

- [ ] Supabase Auth habilitado (email/password)
- [ ] Página de registro funcional (`/auth/signup`)
- [ ] Página de login funcional (`/auth/login`)
- [ ] JWT tokens gerados automaticamente após login
- [ ] Middleware de proteção de rotas (redirect para login se não autenticado)
- [ ] Logout funcional
- [ ] Email de confirmação (opcional para MVP)

## Implementation Notes

Stack:
- Supabase Auth (built-in)
- Next.js App Router
- React Context para state global
- Protected routes via middleware

Flow:
1. User clica "Registrar"
2. Form valida email + password
3. Supabase cria user + envia JWT
4. App redireciona para dashboard
5. Dashboard só acessível com JWT válido
